# Local Food Wastage Management System

## Project Overview

The Local Food Wastage Management System is a Streamlit and MySQL based data analytics application developed during the LABMENTIX Data Analytics Internship. The project helps connect food providers and receivers while analyzing food donations, claims, and distribution patterns.

## Technologies Used

* Python
* Streamlit
* MySQL
* Pandas
* Plotly

## Features

* Interactive Dashboard
* Food Listings Management
* Claims Analysis
* Exploratory Data Analysis (EDA)
* SQL Analysis (15 Queries)
* CRUD Operations
* CSV Export Functionality
* Professional Sustainability-Themed UI

## Database Tables

* Providers
* Receivers
* Food Listings
* Claims

## Project Modules

1. Dashboard
2. Food Listings Explorer
3. Claims Analysis
4. EDA Visualizations
5. SQL Analysis
6. CRUD Operations

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
